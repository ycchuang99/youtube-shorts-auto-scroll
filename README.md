## Install

### Chrome/Edge

1. Open Chrome and go to [youtube-shorts-auto-scroll](https://chromewebstore.google.com/detail/youtube-shorts-auto-scrol/ejiondkplaoolmmnepmpipdpaeibfje?hl=zh-TW)
2. Click "Add extension"
3. Enjoy your YouTube shorts video
